What Are Notes?

Notes let you add comments, instructions, and context to your Shopify data that your whole team can see. Think of them as sticky notes attached to your store records.

  • "Customer requested gift wrapping" → attached to an Order
  • "Restock by Friday, supplier: ABC Inc." → attached to a Product
  • "VIP customer, always offer free shipping" → attached to a Customer
  • "Weekly team meeting agenda" → standalone note (not attached to anything)

Creating a Note

Click Notes in the sidebar
Click the Create Note button (top right)
Choose an Entity Type: Order, Product, Customer, or None (standalone)
If you chose an entity type, select a specific record from the dropdown
Add an optional Title and write your Content using the rich text editor
Click Save

Rich Text Formatting

The note editor supports:

  • Bold and Italic text
  • Bullet lists and numbered lists
  • Blockquotes
  • Code blocks

Attaching Files

You can upload files to any note — great for images, PDFs, or documents.

When creating or editing a note, scroll down to File Attachments
Click Upload File or drag & drop files into the upload area
Click Save to confirm

Supported files: PNG, JPG, GIF, WebP, PDF, DOC, DOCX, TXT, XLS, XLSX, CSV

Viewing & Editing Notes

The Notes page shows all your notes in a table with title, content preview, entity info, author, and date. Click any note to open its detail view.

Pinning Notes

Pin important notes so they always appear at the top of your list. Click the 📌 icon on any note to pin/unpin it.

Pin notes for ongoing issues, reminders, or important customer instructions.

Converting Notes to Tasks

Turn notes into actionable tasks with one click:

Select one or more notes using the checkboxes
Click Convert to Tasks
Review the task preview — the note title becomes the task title, and the content becomes the description
Click Confirm
Task Fields

Converted tasks include title, rich-text description, priority level, status, assignee, and due date offset.

Track on Dashboard

Converted tasks appear in the "Today's tasks" dashboard card alongside manually created and workflow-generated tasks.
