Tasks

Last updated on 1 April 2026

Track what needs to be done, who's responsible, and when it's due. Create tasks manually, from notes, or automatically via workflows.

Creating a Task

Click Tasks in the sidebar
Click the Create Task button
Fill in the Title, choose a Priority, set a Due Date, and optionally Assign To a team member
Add a Description with detailed instructions (supports rich text)
Click Save

Your admin can create custom statuses in Settings (e.g., "Waiting for Customer", "Under Review").

Table View

The default view shows tasks in a table with columns for title, status, priority, assignee, due date, and created date. Click column headers to sort.

Kanban Board View

Switch to the Kanban view for a visual board — tasks appear as cards organized by status. Drag and drop cards between columns to change their status.

💡
To set the default view for your store, go to Settings → Task Preferences → Default View.

Task Timeline

Every task has a timeline that shows its full history — when it was created, every status change, every edit, and if it was created by a workflow (shows which workflow and what triggered it).

Linking Tasks to Notes

Tasks and notes can be connected. When creating a task, search for existing notes to link in the Notes section. You can also convert notes to tasks from the Notes page.

Auto-Created Tasks (From Workflows)

When a workflow has a "Create Task" action, tasks are created automatically. These tasks show an "Auto-created" badge and include a timeline entry showing which workflow created them.
